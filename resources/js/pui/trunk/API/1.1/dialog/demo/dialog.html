对话框实例

alert------PW.mod.Dialog.alert(str,callback,{settings})

  • str:第一个参数，提示的内容（字符串）
  • callback:第二个参数，回调函数
  • {settings}:第三个参数，settings额外配置
  • 返回对话框唯一ID

confirm------PW.mod.Dialog.confirm(str,okCbk,cancelCbk,{settings})

  • str:第一个参数，提示的内容（字符串）
  • okCbk:第二个参数，OK的回调函数
  • cancelCbk:第三个参数，cancel的回调函数
  • {settings}:第四个参数，settings额外配置
  • 返回对话框唯一ID

prompt------PW.mod.Dialog.prompt(tip,orginalTxt,okCbk,{settings})

  • tip:第一个参数，提示的内容（字符串）
  • orginalTxt:第二个参数，初始文本
  • okCbk:第三个参数，OK的回调函数
  • {settings}:第四个参数，settings额外配置
  • 返回对话框唯一ID
